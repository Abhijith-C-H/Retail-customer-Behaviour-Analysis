# Retail-customer-Behaviour-Analysis
End-to-end Business Intelligence project analyzing 3900 customer records to understand shopping patterns. Built with Python for data cleaning, PostgreSQL for analysis, and Power BI for interactive visualization. Delivers actionable insights and recommendations for a leading retail company.

## Project Overview
This project answers the key business question:  
**“How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing and product strategies?”**

## Deliverables Completed
- **Data Cleaning & Modeling**: Python (Jupyter Notebook) – Handling missing values, feature engineering (`age_group`, `purchase_frequency_days`), column standardization, and loading to PostgreSQL.
- **Data Analysis**: PostgreSQL – 10 insightful SQL queries covering revenue by gender/subscription/age group, discount impact, top products, loyalty segmentation, and shipping comparison.
- **Visualization**: Power BI – Interactive dashboard with KPIs, demographic breakdowns, product performance, discount analysis, and dynamic filters.
- **Documentation**: Professional report and presentation summarizing findings and actionable recommendations.

## Tech Stack
- **Python** (Pandas) – Data cleaning and transformation
- **PostgreSQL** – Structured data analysis
- **Power BI** – Interactive dashboard and visualizations
- **SQLAlchemy** – Database connectivity

## Key Insights
- Males contribute significantly higher revenue than females
- Subscribed customers show higher average spend and loyalty
- Middle-aged and Senior customers drive the majority of revenue
- Discounts strongly influence purchase behaviour in specific categories
- Repeat buyers are more likely to subscribe
